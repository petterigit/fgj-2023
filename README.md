## Quick start

Prerequisites

-   [Node](https://nodejs.org/en/)
-   prettier

Install with

```
yarn
```

Run with

```
yarn dev
```

Run with devtools enabled

```
yarm dev:utils
```

Run TSC to check for errors

```
yarn tsc:check
```

Preview build

Runs both build & preview so you can be sure the project compiles & runs

```
yarn preview
```
